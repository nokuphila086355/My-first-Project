# My First Project - C# OOP

A comprehensive learning project demonstrating Object-Oriented Programming (OOP) principles in C#.

## 📚 Project Overview

This project is organized to showcase the four fundamental pillars of Object-Oriented Programming:
- **Encapsulation** - Data hiding and access control
- **Inheritance** - Code reuse and hierarchical relationships
- **Polymorphism** - Multiple implementations and method overriding
- **Abstraction** - Hiding complexity and defining contracts

## 📁 Project Structure

```
My-first-Project/
├── Encapsulation/          # Data hiding, getters/setters, access modifiers
├── Inheritance/            # Base classes, derived classes, code reuse
├── Polymorphism/           # Method overriding, interfaces, virtual methods
├── Abstraction/            # Abstract classes, interfaces, contracts
└── README.md               # Project documentation
```

## 🎯 Key Concepts Demonstrated

### Encapsulation
Protects internal object state by using private fields and public properties/methods to control access to data.

### Inheritance
Allows classes to inherit properties and methods from parent classes, promoting code reuse and establishing hierarchical relationships.

### Polymorphism
Enables objects to take multiple forms through method overriding and interfaces, allowing the same method name to behave differently based on context.

### Abstraction
Hides implementation details and defines interfaces through abstract classes and interfaces, reducing complexity and defining clear contracts.

## 🚀 Getting Started

### Prerequisites
- [.NET SDK](https://dotnet.microsoft.com/download) (latest version)
- C# knowledge (beginner to intermediate)
- A code editor (Visual Studio, Visual Studio Code, or similar)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nokuphila086355/My-first-Project.git
cd My-first-Project
```

2. Build the project:
```bash
dotnet build
```

3. Run the project:
```bash
dotnet run
```
